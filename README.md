# Hatio FTE Assignment

## Todo App with Auth and Gist Support

### Tech Stack

- Frontend
  - Next Js + Typescript
- Backend
  - Nest Js + Typescript + PostgreSQL + Prisma 


## Run Locally


- Database
```bash

 docker compose up

```


- Frontend
```bash

 cd client
 pnpm install
 pnpm run dev

```

- Backend
```bash

 cd server
 pnpm install
 pnpm run start:dev

```
